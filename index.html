About me

Summary

I am a software engineer & front-end architect from Munich, Germany, specialised in building (mobile) web applications with HTML, CSS and Javascript.

During my training for becoming an computer specialist for application development at EDV-Schule Plattling I discovered my passion for building web applications. Since then I've worked for multiple companies as a Python web developer as well as a front-end architect.

Besides my workaday life I'm organising the user group RefreshMunich together with friends, working with a friend on an own product and attending as well as speaking at various web related conferences and user groups.

In short: I like to be surrounded by people who are better then me and learn from them.

My Top Skills

  • Full featured mobile apps with web technologies
