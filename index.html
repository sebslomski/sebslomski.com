Eine ganz heisse Nummer Promotion App

To promote their new cinema movie "Eine ganz heisse Nummer" during the October Fest, Universum Film entrusted cwerk to build a cross platform app for mobile phones.

To achieve the cross platform requirement, the app was built with the web technologies HTML5, CSS3 and JavaScript. The device camera was made accessible through Phonegap.

Together with the designer Dana Nedamaldeen, we created a unique custom user experience, far beyond the at this time typical jQueryMobile-iOS-like user interfaces.

The app was done under strong time pressure, it took less then three weeks from concept to deployment with only one designer and one developer.

Striker Tool

Striker Tool is a custom customer relation management (CRM) solution for RedBull, built on top of catenate's MASIS JSON API.

Contemplated to assist the RedBull salesman during their customer visits, Striker Tool was designed to run on mobile devices, primarily the BlackBerry Torch and the iPad. To achieve this platform independence the tool was built with the web technologies HTML5, CSS3 and JavaScript, back then cutting edge technologies.

Striker Tool is in operations in Germany since July 2011. In due to its popularity and straightforwardness it is going to be internationalized and rolled out worldwide in 2012.

This project was my first project at catenate and also my first JavaScript project. I designed and built the software architecture in less then 5 months. Until now, Striker Tool is the biggest project I've created yet.

About me

Summary

I am a software engineer & front-end architect from Munich, Germany, specialised in building (mobile) web applications with HTML5, CSS3 and Javascript.

During my training for becoming an computer specialist for application development at EDV-Schule Plattling I discovered my passion for building web applications. Since then I've worked for multiple companies as a Python web developer as well as a front-end architect.

Besides my workaday life I'm organising the user group RefreshMunich together with friends, working with a friend on an own product and attending as well as speaking at various web related conferences and user groups.

In short: I like to be surrounded by people who are better then me and learn from them.
